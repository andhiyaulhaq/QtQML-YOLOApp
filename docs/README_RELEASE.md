# YOLOApp (QtOpenCVCamera)

YOLOApp is a high-performance desktop application for real-time object detection, pose estimation, and image segmentation.

## 🚀 How to Run

1.  Navigate to the application folder.
2.  Double-click `appCamera.exe` to launch the application.

## 🛠️ Requirements

- A Windows 10/11 64-bit system.
- The required libraries are included in the `libs/` folder.
- Ensure your camera is connected and accessible.

## 🎮 Features

- **Object Detection**: Detect common objects in real-time.
- **Pose Estimation**: Track human skeletal poses.
- **Image Segmentation**: Identify and mask objects with pixel precision.
- **Real-time Performance**: Optimized for smooth interaction and low latency.

## 📝 Note

If you encounter any issues, ensure all `.dll` files in the `libs/` subdirectories are present.
